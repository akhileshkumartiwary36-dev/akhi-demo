## 🎯 Topics Covered

- **Algorithm Basics**: GCD, Fibonacci, Factorization, Tower of Hanoi
- **Searching Algorithms**: Linear Search, Binary Search, Pattern Matching
- **Sorting Algorithms**: Selection Sort, Merge Sort, Quick Sort, Insertion Sort
- **Non-Comparison Sorting**: Counting Sort, Bucket Sort, Radix Sort
- **Greedy Algorithms**: Fractional Knapsack, Huffman Coding
- **Graph Algorithms**: Dijkstra's Algorithm, Minimum Spanning Tree (Prim & Kruskal)
- **Time Complexity Analysis**: O(1), O(n), O(log n), O(n log n), O(n²), O(2^n)

## 📊 Complexity Summary

| Complexity | Example Algorithms |
|------------|-------------------|
| O(1) | Direct Access, Constant Operations |
| O(log n) | Binary Search, Fibonacci (De Moivre) |
| O(n) | Linear Search, Array Traversal |
| O(n log n) | Merge Sort, Quick Sort, Dijkstra |
| O(n²) | Selection Sort, Bubble Sort, Prim MST |
| O(2^n) | Fibonacci (Recursive), Tower of Hanoi |
